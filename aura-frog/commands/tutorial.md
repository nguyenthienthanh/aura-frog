# Command: tutorial

**Category:** Onboarding
**Syntax:** `tutorial`

---

## Purpose

Interactive walkthrough of your first Aura Frog workflow.

---

## Response Template

```
Welcome to Aura Frog!

1. **Read the tutorial:** `docs/guides/FIRST_WORKFLOW_TUTORIAL.md`
   → Walks you through a complete 5-phase workflow step by step

2. **Or try it now:**
   → Type `workflow:start Add a simple validation to any form`
   → I'll guide you through each phase

3. **Quick reference:**
   | Action | Command |
   |--------|---------|
   | Build feature | `workflow:start "task"` |
   | Fix bug | `bugfix:quick "desc"` |
   | Approve phase | `approve` |
   | Check status | `workflow:status` |

Need help? See `docs/TROUBLESHOOTING.md` for common issues.
```
