# Command: agent:activate

**Category:** Agent Management  
**Syntax:** `agent:activate <agent-name>`

---

## Description

Manually activate a specific agent for the current workflow.

---

## Usage

```
agent:activate tester
agent:activate frontend
```

---

## Output

```markdown
✅ Agent Activated: tester

The QA Automation agent is now active and will participate in:
- Test planning
- Test implementation
- Coverage analysis
- QA validation

This agent will remain active for the current workflow.
```
